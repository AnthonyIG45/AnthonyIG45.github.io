# AnthonyIG45.github.io
Javascript app to calculate potential and current grades for school work pages.

Goals:
1. Manually enter a number value - Weight and score - and have calculate what percentage you got
2. Dynamic grade value enter to determine overall class grade
3. Comparison between current total (if not done with a class) and overall total grade percentage
4. Potential grade calculation (good for estimating needed scores for certain grades)
5. Media upload of scores and weights through image or pdf
6. Image processing of scores to automatically acquire grades and weights from syllabus/gradebook
7. Fully automatic or manual processing of grades and prediction grades
